Alur Pengambilan Key
1
Jalankan script di executor Roblox. GUI verifikasi muncul di layar.
2
Klik tombol "Belum punya key? Ambil di sini" di dalam GUI.
3
Script otomatis request signed token dan link disalin ke clipboard.
4
Buka link di browser. Key otomatis tergenerate dengan HWID dan username dari game.
5
Salin key, paste ke GUI di game, tekan Verifikasi.
